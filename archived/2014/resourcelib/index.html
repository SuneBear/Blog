如何建立个人资源库 – 印象大脑

很早就想分享一下如何建立个人资源库（Personal resource library），我称这个库为印象大脑（源自印象笔记），它可以永久的记住所有资源，侧重资源的统一性。

印象大脑是什么？

首先用一个等式来说明：
印象大脑 = 记事本 + 收藏夹 + 文件 + 内容分享

  1. 1.记事本：记录私密文字的地方，用软件实现，如EverNote
  2. 2.收藏夹：收集喜欢网址的地方，如liebao
  3. 3.内容分享：博文或者社交网络发表的内容
  4. 4.文件：各种在存储设备的数据，在硬盘，U盘，网盘，手机等地方

当这四个子脑都在云中，并用适合的自己方式组织，那么将会诞生一个非常好用且完整的印象大脑

子脑：记事本

一定要使用具有云功能的记事软件，这样可以避免记录不同步，或者记录分散。
当选择好软件后，接下来就是构建框架，输入记录，分类记录。
我的记事本框架含有：便笺，GTD，时间轴，账号资源，经验总结，普通格式文章等。
记事本一定要是自己原创的私密内容，互联网发现的内容放到收藏夹。
说明：框架并不是软件的功能，而是自己根据需求规定的一种格式，可参见用笔纸写GTD。

子脑：收藏夹


传统的收藏夹实质上是构建网址导航，根据自己的喜好来收藏排序。
我这里说的收藏夹在此基础上多了以下功能：

  1. 1.缩略图列表：可自己上传缩略图，默认是网站截图
  2. 2.文字节选：对选择的区域高亮显示，且自动定位
  3. 3.高级搜索：除了对标题，网址搜索外，还可以对缓存内容进行搜索
  4. 4.生成网址导航：对点击次数多的网站排序，分类网址，有些类似九宫格与hao123

这样的收藏夹将可以记录几万网址，并且结构清晰，使用方便。

子脑：内容分享

内容分享是与外界交流做出的反馈，因为我们不能只获取。
你现在看到的这篇博文就是内容分享，你在微博，QQ，邮箱发布的内容都叫内容分享。
内容分享可以是原创，也可以是转载，但这都将影响接收者对你的反馈做出评级。

在互联网没有隐私可言，我们的一言一行都将会成为评判我们的依据，多分享有价值的内容，多表达自己，不然只能埋怨世界不理解你。

子脑：文件

这里说的文件指一切存在存储设备的数据，其他三个子脑实质上也是数据。

文件是笼统的叫法，它可以是使用的软件，也可以是制作的文档，拍摄的图片。
组织文件的方法有两种，一种是物理式的通过磁盘目录分放，一种是逻辑式的通过特征筛选。
具体怎么组织文件，可以去谷歌搜索。

如何开始组建印象大脑？

其实我们一直都在组建印象大脑，我只是提出了一个概念，有了概念才能形成体系。
另外一定不要按照我以上的方法组建自己的印象大脑，那并不适合你。
组建适合自己的印象大脑需要几年的时间才能成熟，他的服务对象是真正的大脑。

适合这个关键词很重要，千万不要被印象大脑给限制了，切记。
