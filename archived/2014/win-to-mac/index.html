前端工程师Win转Mac的初体验

作为一个多年的Win系统用户，熟悉了各种快捷键，设置，命令，并收集了各种软件与游戏。已下定决心和Win成为终身伴侣，就好像许多人依旧不舍XP一样，那种操作的快感是新系统无法给予的。不过在生命的1/3阶段，我突然成为了异教徒，我刚进入的那家公司所有人都用Mac OS，为了不被烧死，我必须得改信仰了。以下就是更改信仰的记录与指南。

起初我是被逼的，后来我很享受

首先说说快捷键，Mac下没有Win，Home，End，PageUp&Down。Control两个系统虽然名字一样，但其实Mac下的Command更像Win的Control。最不适应的还是Win＋E，Win＋D没有了，Command位置和Win系统下的Alt位置一样，复制粘贴很不习惯。所以我就查了常用的快捷键，用了几次是就适应了Command，连Command都适应了，再把常用快捷键记下来，快捷键就过关了，如果熟悉Chrome的快捷键，那么上手更快。最后对快捷键的总结就是老老实实记住它，不要多想。

再来说说使用习惯，没有了任务栏，但是有了顶部状态栏，对于智能手机不陌生对我们，也很快就能适应。还有一点我觉得非常棒，所有软件的菜单都在状态栏上面，这样形成了统一。最底部的悬停浮动条，可以快速进入软件，感觉就是取代了桌面，软件之间的切换也不用通过状态栏或者回桌面，直接Command＋Tab，同时用触摸板进行快速定位。还有很多功能基本都是直接上手，完全不用思考的，这个不是Mac的功劳，而是Mac与Win有很多类似的地方。

触摸板与键盘是非常好用，这样就可以不用连接任何外设，让MacBook完全独立，方便移动。触摸板有十几个手势，记下来后比鼠标还要好用，但如果是作图或者对指针移动频率高，还是鼠标好，毕竟板子摸长了手酸。我用了两个星期没有关过机，也不会主动关闭软件，也不用担心性能问题，放心且高效的处理事物。还有最后一点，电脑外观与系统外观都很美，创造一种沉浸式的用户体验，对于创造者来说，更容易创造卓越的东西。

我以为操作习惯很难改，其实两天时间就够了，所以不要畏惧新的东西。
Win与Mac可以同时使用，他们各有优点，不要有那种只用Mac的心理。

扯淡之余是干货

常用快捷键列表

切换应用 Command + Tab //可同时用触摸板或者Esc
切换窗口 Command + ~ //波浪号在Tab上面
设置应用 Command + , //逗号
切选项卡 Control + Tab
切输入法 Command + Space //空格
显示桌面 Command + F3
截图工具 Command + Shift +3 Command + Shift + 4 //3是全屏
关闭窗口 Command + W
新建窗口 Command + T
复制数据 Command + C
粘贴数据 Command + V //加上Option就成了剪切
重命名 Enter //而不是Win下的F2
=============Sublime Text=============
跳到行首 Command + Left
跳到行尾 Command + Right
跳到页首 Command + Up //Fn + Left
跳到页尾 Command + Down //Fn + Right
上一页 Fn + Up
下一页 Fn + Down
多列选择 Option + Left Mouse
选择重复 Command + D
