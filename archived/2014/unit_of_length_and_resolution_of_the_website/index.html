网站常用长度单位与分辨率你真的懂吗？

某一天sune被一些长度单位给迷惑住了，浪费了很多时间。于是决定把它们都搞懂，以免下次再遇到。主要提到px，em，%以及分辨率，因为这几个在网站中常用到。pt，in，cm这几个是在打印时用到，所以简单的说明一下。最主要的是仅仅这一篇文章就可以让你了解网站里面的常用单位。

px

px是什么

px就是像素（Pixel），位图成像的点。这个点的大小取决于分辨率。
当然这个点并不是实际存在的，这个点需要依赖硬件设备来展示。比如LED显示屏，可以用红绿蓝三个发光二极管来展示一个像素点。

px作为单位时

当px作为单位时，就不要再理解px为点。此时此刻它已经是一个长度单位了，再也回不去了~~
它是一个相对长度单位，也就是具体自然长度取决于显示屏的分辨率。
在具体网站建设中，如果你不是做一个响应式网站，那么只用px就可以了。

px单位怎么用

px使用很简单的，直接在数值后面加上”px”两个字母。
难点在于取值，取一个什么样的值让网页最美观。
具体取什么值这个要重新写一篇文章才能说清楚，经验很重要，要多多去观察优秀的网站。目前常见网站的宽度都为960px~1000px之间。如果网站太宽造成整个屏幕太饱满，会分散注意力。

px举个栗子

font-size:12px;     /*文字大小为12px*/

width：200px;       /*元素宽度为200px*/

margin-left：-12px;  /*向左移动12个px，原本是离左12个px，但是加了一个负号就相反了*/

........            /*像素的使用很简单，难点在于取值，取一个什么样的值让网页最美观*/

em

em是什么

em是一个以字体高度为长度的单位，是一个相对长度单位。它比px更加的灵活，因为em的最终长度决定与父级元素的字体高度。通常情况下，浏览器默认的父级字体大小为16px，那么我们可以根据16px来进行取em的值，这个时候2em=32px，0.625em=10px。但是一旦父级元素的字体高度改变了，那么用em的元素单位的长度都会改变，所以用em时不要轻易改变父级元素的字体高度。

em怎么用

em一般用来作为字体大小、行高、间距等的单位，很少用来作为 盒子的单位。
其实在中国98%的网站都没有用em，因为em比px要复杂，可以说em就是一个px的倍数关系，最终用到元素上面，依旧是px。

但是现在因为很多响应式网站，所以才有了em的用武之地。因为在自适应网站里面有一个判断语句，可以在不同的设备下，给父级元素高度赋不同的值

h1 {

  font-size: 3em;    /* 3em=48px 所以这个元素的字体大小为48px  */

  line-height: 1;   /* 这个地方直接省略了em，但是浏览器会自动识别，行高为16px  */

  color: #222;     

  margin-bottom: .5em;  /*与下面的外边距为0.5em = 8px 这个地方省略掉了0，但是浏览器会自动识别  */

  float: left;

  width: 100%;        /*100% 具体的值也与父级元素有关*/

  font-family: "microsoft yahei";

  letter-spacing: 3px;  /* 字间距为3px*/

}

%

%是什么

它就是百分比…这个没有什么好解释的。也是一种相对长度单位，但这里要区别一下，px是的值是不受父级元素的影响的，但是em与%都要受父级元素的影响。说px是相对长度单位，是因为受分辨率的影响。

%怎么用

在网页设计中，%一般用来作为盒子与包裹的单位。假设我们需要让某一个图片全屏展示，但是在不同的设备中，尺寸不一样，所以如果用px的话就固定了，会导致无法全屏或者设置过小。如果用%，设置width:100%;height:100%。这样就可以在任何设备中全屏显示，包括手机。

.clear_bg {  /*这个就是我博客背景的样式，你看看是不是全屏*/

width: 100%;  /*宽度100%*/

height: 100%;  /*高度100%*/

z-index: -9998;  /* z轴坐标为-9998 */

}

分辨率

分辨率有很多种，这里我就讲像素分辨率（ppi）。像素分辨率就是指每英寸所含的像素点数目，我们的手机，电脑所说的分辨率都是像素分辨率。

我们的日常生活中都会有一个误解，会把宽度与高度给误认为是分辨率。常见有：某个手机厂商说自己屏幕的分辨率1136*640，屏幕尺寸为4英寸。1136*640并不是分辨率，而是高度与宽度。但是这样说并不是完全错误，因为分辨率由尺寸与宽高来决定，这个手机的分辨率为325.97ppi，算是非常棒了。手机厂商用宽高来代替分辨率，这样使表达更直白易懂，不过也有手机厂商理由宽高当分辨率耍猫腻，这就不多说了。我们电脑屏幕的分辨率一般为96ppi，而不是什么1024*768…

pt,in,cm

这三个单位是绝对单位长度，它们的自然长度是固定的，除非你离开地球，超过光速…

pt是磅=1/72英寸，in是英寸，cm是厘米。绝对长度单位是用在印刷行业的。如果你的网站有打印功能，那么可以把打印的东西设置成绝对长度单位。

比如现在网络打印电子卷，电影票，就可以用这些单位。

总结

说了这么多的单位，或许大家会有些糊涂了，糊涂就对了…我就来进行一下简单的总结吧。

在网页设计中用px与%就可以应付一切情况，其他的单位可以不管。了解其他单位有助于思路扩展。px既可以是点，也可以是单位，具体是什么，要看具体情况。未来我将会专门写一篇《像素与百分比取值的艺术》，了解单位取值的奥妙。

好了，到这里就结束了，如果没有看懂，这个很正常，我表达能力太屌。有疑惑，问问度娘，看看手册，实战操作……谢谢你的阅读。
